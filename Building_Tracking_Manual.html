1. Overview

The Building-Wise Tracking tab gives your team a real-time view of every precast panel across every building — who is casting it, which building it belongs to, and whether it has been completed.

Your project contains two types of panels:

Type Description Who casts?
In-House Panels scheduled and cast in Capriole's own casting yard Capriole production team
By Others Panels originally assigned to an external sub-contractor External contractor or Capriole (if reassigned)
ℹ️

Important: In some situations, panels originally labelled By Others may end up being cast by Capriole's own yard instead. The tracking tab lets you record this without changing the original contract data.

1.1 Status States

Each building cell (B1, B2, B3 …) for every panel type can be in one of three states:

–
PendingNot yet cast · grey dash
✓
Cast (By Others)External contractor confirmed cast · green
✓IH
Cast In-HouseCapriole cast this instead · blue
✅

In-House panels (green rows) are updated automatically by the engine when a day is marked complete in the Order Schedule. You do not need to click them manually.

1.2 Who Should Update This Tab?

  • Site Engineer / Production Supervisor — marks By Others panels after confirming cast with the sub-contractor
  • QC Team — verifies and counter-checks cast counts against delivery notes
  • Project Manager — monitors overall % completion via the Print Report
🚫

Do not mark a panel as cast before physical confirmation. The system has no undo — an incorrect mark must be clicked again to cycle back to Pending.

2. Opening the Building Tracking Tab

  1. Open the project in the Precast Engine.
    Sign in to the cloud from the home screen and click your project card, or open a local .pfproj file.
  2. Click the Building Tracking tab in the top navigation bar.
    It is the second tab from the right, labelled 🏢 BUILDING TRACKING.
  3. Select the correct floor using the floor tabs (e.g. FF, GF) shown below the title bar.
    Each floor is tracked independently.

3. Updating Cast Status

3.1 Marking a "By Others" Panel as Cast

When a sub-contractor confirms that a panel has been cast, follow these steps:

  1. Navigate to Building Tracking and select the correct floor tab (FF or GF).
  2. Find the panel row. By Others rows have an amber/cream background and show a red "By Others" source badge.
  3. Locate the building column (B1, B2, B3 …) for the specific building that has been cast.
  4. Click the cell once. It will turn ✓ IH (blue) — meaning Cast In-House by Capriole.
  5. Click again to advance to ✓ (green) — meaning Cast by the external contractor (By Others).
  6. Click a third time to return to – (Pending) if you made an error.
⚠️

Choose the correct state:
Use ✓ IH (blue) when Capriole's own casting yard produced the panel.
Use ✓ (green) when the external sub-contractor produced the panel as per contract.

3.3 Marking In-House Panels as Cast

In-House panel cells in the Building Tracking table are read-only — you cannot click them directly here. Their status is driven by the Order Schedule tab. Follow these steps to mark an In-House panel as Cast:

  1. Click the ORDER SCHEDULE tab in the top navigation bar.
  2. Use the Filter Bar at the top of the table to narrow down the list:
    — Type the panel code (e.g. GF-WL-01) in the Search box, or
    — Click a Family chip (WL, IP, WN, PB) to filter by family, or
    — Click a Building chip to show only one building.
  3. Find the correct panel row. The row shows: Date · Table · Type · Building · Dimensions · Weight · Mould · Status.
  4. In the Status column, click the dropdown and change it from Pending to ✓ Cast.
    The row background turns green immediately to confirm.
  5. Return to the Building Tracking tab — the corresponding building cell will now show a green ✓.
  6. Click 💾 Save Patch in the top-right to sync changes to the cloud.

3.4 Auto-Save

Every click is saved automatically — there is no Save button needed. The status is stored in the project file. The next time you open the project, all statuses will be exactly as you left them.

💡

After marking panels, click 💾 Save Patch in the top-right corner to sync your updates to the cloud. This ensures all team members see the latest status when they next open the project.

4. Generating Reports

4.1 Print Report (PDF)

Click the 🖨 Print Report button in the top-right of the Building Tracking tab. A new window will open with a formatted report that includes:

  • Capriole logo and project details in the header
  • Overall cast progress percentage
  • SVG donut charts — one per floor showing % completion
  • Horizontal bar chart showing progress per building
  • Full panel tracking table with colour-coded cells
  1. Click 🖨 Print Report. A print preview window will open automatically.
  2. In your browser's print dialog, set Paper size: A3 and Orientation: Landscape.
  3. Enable "Background graphics" (or "Print backgrounds") so the coloured cells print correctly.
  4. Select Save as PDF as the destination to generate a PDF file.
  5. Share the PDF with stakeholders via email or WhatsApp.
⚠️

The report only shows the currently selected floor. If you need reports for both FF and GF, print once per floor by switching the floor tab between prints.

4.3 Switching Between Floors

Use the floor tabs (FF / GF) immediately below the title. The selected floor tab will be highlighted. The Print Report and Export CSV will use the currently active floor.

⚡ Daily Workflow Checklist
  1. Open project from cloud or local file
  2. Click Building Tracking tab
  3. Select the correct floor (FF / GF)
  4. For each confirmed cast — click the building cell once or twice
  5. Verify Cast count matches your daily report
  6. Click 💾 Save Patch to sync to cloud
  7. Print or export if a progress report is needed
🚫 Common Mistakes to Avoid
  • Marking a panel cast before physical confirmation
  • Forgetting to Save Patch after updating (changes won't sync)
  • Clicking In-House cells (they are read-only)
  • Printing without switching to the correct floor tab first
