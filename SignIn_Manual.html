1. Why Sign In to the Cloud?

The Precast Engine stores every project as a patch file on Capriole's private ownCloud server (cloud.capriolegroup.org). Signing in lets you:

FeatureWithout Sign-InAfter Sign-In
View running patches❌ Not visible✅ All patches load on home screen
Open a projectLocal files only✅ One-click open from cloud card
Save progressLocal file only✅ Auto-saved to cloud for team access
Team collaboration❌ Not possible✅ Everyone sees the same live data
ℹ️

The engine uses ownCloud App Passwords — not your regular ownCloud login password. An App Password is a special one-time credential you generate yourself inside ownCloud. This keeps your main account secure.

1.1 What You Will Need

  • Access to cloud.capriolegroup.org (ask your IT admin if you don't have an account)
  • Your ownCloud username
  • An App Password generated from ownCloud Settings (see Section 2)
  • Internet connection

1.2 Supported Platforms

The Precast Engine runs in any modern web browser. Recommended browsers:

✅ Google Chrome
✅ Microsoft Edge
⚠ Firefox (may block popups)

2. Create an App Password in ownCloud

You must do this once before you can sign in to the engine. Your App Password is a 19-character code in the format xxxx-xxxx-xxxx-xxxx.

  1. Open your web browser and go to:
    cloud.capriolegroup.org
    Log in with your normal ownCloud username and password.
  2. Click your profile picture or initials in the top-right corner of the screen.
  3. From the dropdown menu, click Settings.
  4. In the left sidebar, click Security.
  5. Scroll down to the "App passwords / tokens" section.
  6. In the App name field, type: Precast Factory
    Then click Create new app password.
  7. ownCloud will display a password in the format xxxx-xxxx-xxxx-xxxx.
    Copy it immediately — it will only be shown once.

3. Sign In to the Precast Engine

Once you have your ownCloud username and App Password, follow these steps to connect:

  1. Open the Precast Engine in your browser and wait for the home screen to load.
    You will see the message: "Not signed in — 🔐 Sign In to connect to your cloud running patches"
  2. Click any of the following buttons — they all open the same sign-in dialog:
    • 🔐 Sign In to Cloud — large button in the top-right of the home screen
    • 🔐 Sign In — small button in the Running Patches section header
    • 🔐 Sign In underlined link in the status text

5. Troubleshooting

5.1 Common Error Messages

Error MessageCauseFix
✗ Auth failed — check username / app password Wrong username or App Password entered Re-open Sign In dialog and re-enter both fields. Make sure to use your App Password, not your main ownCloud password
Failed to load — HTTP 404 Project folder not found on server Contact your administrator — the cloud folder may have been moved or renamed
⚠ Offline — showing cached patches No internet connection or server temporarily unavailable Check your network connection and click 🔄 Refresh to retry
✗ Network error / Failed to fetch Browser blocked the request (CORS or firewall) Use Google Chrome or Microsoft Edge. Contact IT if the issue persists

5.2 App Password Expired or Lost

  1. Go to cloud.capriolegroup.org and sign in with your main password.
  2. Navigate to Settings → Security → App passwords / tokens.
  3. Find the Precast Factory entry and click the 🗑 Delete (bin) icon next to it.
  4. Create a new App Password following the steps in Section 2.
  5. Return to the Precast Engine, click 🔐 Sign In, and enter your username and the new App Password.

5.3 Re-authenticate (Quick Fix)

If you see a red error on the home screen with a Re-authenticate link, click it directly — it opens the Sign In dialog immediately without navigating away.

⚠️

If you switch computers or use a private/incognito browser window, you will need to sign in again. Your credentials are stored per browser, not per user account.

5.4 Quick Reference — Sign-In Steps

One-time setup (do once per device)
  1. Log in to cloud.capriolegroup.org
  2. Go to Settings → Security → App passwords
  3. Create password named Precast Factory → copy the code
Every time you open the engine
  1. Click 🔐 Sign In to Cloud on the home screen
  2. Enter your ownCloud username
  3. Paste your App Password (xxxx-xxxx-xxxx-xxxx)
  4. Click Connect →
  5. Your patches appear — click any card to open a project
